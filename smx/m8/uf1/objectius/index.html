UF1: Ofimàtica i eines web

Resultats d'aprenentatge

En finalitzar aquest mòdul l’alumne/a...

  1. Instal·la, si s’escau aplicacions web, descrivint les seves característiques i entorns d’ús.
    1. Descriu l’origen i evolució dels serveis web de darrera generació
    2. Instal·la si s’escau, diferents aplicacions web, descrivint les seves característiques i entorns d'ús.
    3. Configura i utilitza el correu electrònic web. Gestiona comptes d’usuari.
    4. Configura aplicacions de calendari web.
    5. Reconeix les prestacions específiques de les aplicacions instal·lades (cites, tasques, entre d'altres).
    6. Configura les aplicacions web per a integrar-les amb el servei de correu.
    7. Cerca i interpreta documentació tècnica en les llengües oficials i en les de més ús al sector.
    8. Documenta adequadament la configuració de les aplicacions web d'escriptori, les incidències aparegudes i les solucions aportades.
  2. Instal·la aplicacions d’ofimàtica web, descrivint les seves característiques i entorn d’ús.
    1. Cerca i interpreta l’origen i evolució de les aplicacions ofimàtiques.
    2. Estableix la utilitat de les aplicacions d’ofimàtica web.
    3. Descriu diferents eines d’ofimàtica web (processador de textos, full de càlcul, entre altres).
    4. Configura aplicacions d'ofimàtica web.
    5. Gestiona els comptes d’usuari.
    6. Aplica criteris de seguretat en l’accés dels usuaris.
    7. Reconeix les prestacions específiques de cadascuna de les aplicacions instal·lades.
    8. Utilitza les aplicacions de forma col·laborativa.
    9. Documenta adequadament la configuració i gestió de les aplicacions ofimàtiques web, les incidències aparegudes i les solucions aportades.

Continguts

Els conginguts d'aquesta UF són els següents:

  1. Instal·lació s’escau d’aplicacions web
    1. Origen i evolució de les aplicacions web
    2. Instal·lació si s’escau d’aplicacions web. Característiques i entorn d’ús.
    3. Correu electrònic web. Gestió de comptes d’usuari
    4. Configuració de calendari web
    5. Eines específiques web: cites, tasques, entre d’altres.
    6. Configuració de les diferents aplicacions per integrar-les a un servidor de correu
    7. Documentació tècnica
    8. Elaboració de documentacions
  2. Realitza tasques de configuració d’aplicacions d’ofimàtica web:
    1. Origen i evolució de les aplicacions ofimàtiques web
    2. Funcionalitats de les aplicacions d'ofimàtica web Característiques distintives de les aplicacions d'ofimàtica web.
    3. Configuració i si s’escau instal·lació d’aplicacions d’ofimàtica web
    4. Gestió d’usuaris.
    5. Permisos associats als usuaris.
    6. Comprovació de la seguretat.
    7. Utilització de les funcionalitats de les aplicacions d'ofimàtica web.
    8. Utilització col·laborativa de les aplicacions d'ofimàtica web.
    9. Documentació de la configuració, de les incidències aparegudes i de les solucions aportades
